## 📝 使用建议

### 推荐工作流
1. 打开浏览器访问 `http://localhost:8080`
2. **直接输入目标进程PID**（不要点击"加载进程列表"）
3. 点击"✅ 使用此PID"
4. 立即开始调试（秒级响应）

### 获取PID方法
```bash
# 方法1: adb shell
adb shell "ps | grep 进程名"

# 方法2: 在设备上用其他工具查看PID
```

### 避免等待
- ❌ 不推荐: 点击"加载进程列表"（20秒+）
- ✅ 推荐: 直接输入PID（即时响应）
