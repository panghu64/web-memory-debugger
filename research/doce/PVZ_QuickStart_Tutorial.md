#### 一键PowerShell脚本:

```powershell
# pvz_quick_coins.ps1
$amount = Read-Host "输入金币数量 (默认999999999)"
if ([string]::IsNullOrEmpty($amount)) { $amount = "999999999" }

$file = "coins200k0.dat"
$remote = "/data/data/com.ea.game.pvzfree_cn/files/userdata/$file"

Write-Host "📥 导出存档..." -ForegroundColor Cyan
adb pull $remote $file

Write-Host "✏️  修改金币为 $amount" -ForegroundColor Yellow
Set-Content -Path $file -Value $amount -NoNewline

Write-Host "📤 导入存档..." -ForegroundColor Cyan
adb push $file $remote

Write-Host "`n✅ 完成! 启动游戏查看效果!" -ForegroundColor Green
Remove-Item $file
```

### 💰 修改金币 (3分钟)

```

   ├─> 选择全部结果

6️⃣ 查找总金币
   ├─> 记住刚才的地址 (例: 0x12345678)
   ├─> 手动添加地址: 0x12345670 (减8)
   ├─> 类型: QWORD
   ├─> 修改为: 999999999
   └─> ✅ 冻结
```

## 🎯 方法3: 解锁全部商店物品

### 快速方法

```
1️⃣ 记录未购买商品数量
   └─> 例如: 有30个商品未购买

2️⃣ GG搜索状态值
   ├─> 搜索: 1 (未购买状态)
   ├─> 类型: DWORD
   └─> 结果会很多

3️⃣ 购买一个商品
   └─> 该商品状态变为 3

4️⃣ 搜索: 3
   └─> 找到刚购买的商品

5️⃣ 查看地址附近
   └─> 应该是64个连续的int32

6️⃣ 批量修改
   ├─> 长按结果 → 批量编辑
   ├─> 范围: 当前地址 ~ +252 (63*4)
   ├─> 值: 3
   └─> 确认

✅ 全部64个商品解锁!
```

### 精确方法

```
已知: 商店状态数组偏移 = PlayerData + 0x3BB8

步骤:
1. 找到金币管理器地址 (通过搜索金币)
2. 该地址就是 PlayerData
3. 计算: PlayerData + 0x3BB8 = 商店数组起始
4. 手动添加64个地址:
   - 0x3BB8, 0x3BBC, 0x3BC0, ..., 0x3CB4
5. 全部改为 3
```

## 🚀 终极快捷方案

### 一键修改脚本 (GG Lua)

```lua
-- 快速修改金币+解锁商店

gg.toast("🌻 PVZ一键修改")

-- 搜索金币
gg.clearResults()
gg.setRanges(gg.REGION_C_ALLOC)

local currentCoins = gg.prompt({"当前金币:"}, {0}, {"number"})[1]
gg.searchNumber(currentCoins, gg.TYPE_DWORD)

if gg.getResultsCount() < 20 then
    local r = gg.getResults(1)
    local addr = r[1].address
    
    -- 修改金币
    gg.setValues({
        {address = addr - 8, flags = gg.TYPE_QWORD, value = 999999999, freeze = true},
        {address = addr, flags = gg.TYPE_DWORD, value = 999999, freeze = true}
    })
    
    -- 解锁商店
    local items = {}
    for i = 0, 63 do
        table.insert(items, {
            address = addr + 0x3BB0 + i * 4,
            flags = gg.TYPE_DWORD,
            value = 3
        })
    end
    gg.setValues(items)
    
    gg.toast("✅ 完成: 金币+商店解锁!")
else
    gg.toast("结果太多,请手动操作")
end
```

## 🆘 常见问题

### Q1: 找不到金币地址?

**A**: 
```
原因: 搜索范围太大
解决: 
  1. 多次搜索缩小范围
  2. 使用精确搜索而非模糊搜索
  3. 确保游戏在前台运行
  4. 尝试搜索金币±1的值
```

### Q2: 修改后没效果?

**A**:
```
检查清单:
  ☑ 是否已保存修改
  ☑ 是否冻结数值
  ☑ 数值类型是否正确
  ☑ 是否修改了正确的地址
  ☑ 游戏是否重启过 (某些修改需重启)
```

### Q3: 游戏闪退?

**A**:
```
可能原因:
  1. 修改了错误的内存区域
  2. 数值超出安全范围
  3. GG版本不兼容

解决:
  - 减小修改的数值
  - 更新GG到最新版
  - 使用虚拟空间隔离
  - 恢复存档重新开始
```

### Q4: 金币文件修改无效?

**A**:
```
检查:
  1. 文件名是否正确? (coins200k0.dat)
  2. 用户ID是否正确? (0是第一个用户)
  3. 是否有换行符? (应该只有数字)
  4. 是否有多余空格?

正确格式:
  文件内容仅: 999999999
  无换行、无空格、无其他字符
```
